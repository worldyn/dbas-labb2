**what rooms are available for a given time slot**
```SQL
SELECT name
FROM Room
WHERE NOT EXISTS (
  SELECT *
  FROM Booking
  WHERE start < timestamp '2018-02-16 14:00:00'
  AND finish > timestamp '2018-02-16 13:00:00'
  AND Booking.room_id = Room.room_id
);
```
